## Overview

Generate photorealistic or stylized images using text prompts, image references, and adjustable parameters.

## How it works

<CardGroup cols={2}>
  <Card title="Prompt" icon="pen-to-square">
    Describe the scene, subject, and desired style.
  </Card>
  <Card title="Reference" icon="image">
    Optionally guide results with an input image.
  </Card>
  <Card title="Parameters" icon="sliders">
    Control size, style strength, guidance, and steps.
  </Card>
  <Card title="Outputs" icon="download">
    Export as PNG/JPG/WebP; save presets for reuse.
  </Card>
</CardGroup>

## Workflow

- Write a clear prompt with subjects, attributes, and context
- Add a reference image if you need composition or branding consistency
- Tune guidance and style strength to balance creativity and fidelity
- Generate variations and pick the best result

## Tips

- Include lighting, camera, and mood keywords for predictable aesthetics
- Use negative prompts to avoid unwanted artifacts
- Save successful prompts as presets for consistency across campaigns
