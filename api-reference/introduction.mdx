## Overview

Use the API to automate image and video generation, apply upscaling, motion regions, and style changes. Endpoints and schemas are defined in the OpenAPI specification included in this repository.

## Authentication

All endpoints use Bearer token authentication as defined in the spec.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```

## Next steps

- Review the OpenAPI spec in `api-reference/openapi.json`
- Explore endpoint examples in this section
- Generate a token and call endpoints from your workflow
